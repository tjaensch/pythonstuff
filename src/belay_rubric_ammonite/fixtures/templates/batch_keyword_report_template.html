Parses ISO XML metadata record and validates relevant keywords against library of possible NASA Global Change Master Directory (GCMD) keywords. Cosine similarity analysis is performed on invalid keywords to recommend valid phrases. Scores range from 0.0 (completely dissimilar) to 1.0 (a content match). More on applying cosine similarity: cosine similarity for vector space models. For NOAA guidance on GCMD Keywords visit the EDM Wiki Keywords Best Pracices.
